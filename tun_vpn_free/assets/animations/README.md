# Animations

Place Lottie animation JSON files here.

Recommended animations:
- `vpn_connecting.json` - Spinning/loading animation for connecting state
- `vpn_connected.json` - Success animation when connected
- `vpn_shield.json` - Shield pulse animation

You can download free Lottie animations from:
- https://lottiefiles.com/search?q=vpn
- https://lottiefiles.com/search?q=shield
